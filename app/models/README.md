# Direktori Model (Data)

Folder ini berisi class yang merepresentasikan tabel-tabel di database. Menggunakan `Core/Database.php` untuk menjalankan query.

## Daftar Model Utama

- `Anggota.php`: Interaksi dengan tabel `anggota`.
- `Simpanan.php`: Interaksi dengan tabel `simpanan` dan `transaksi_simpanan`.
- `Pinjaman.php`: Interaksi dengan tabel `pinjaman`.
- `Angsuran.php`: Interaksi dengan tabel `angsuran`.
- `User.php`: Interaksi dengan tabel `users` untuk login.
- `SimpananTransaksi.php`: Interaksi dengan tabel `simpanan_transaksi` dan `kas_transaksi` untuk manajemen saldo.
- `Kas.php`: Interaksi dengan tabel `buku_kas`.

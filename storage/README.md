# Direktori Penyimpanan (Storage)

Digunakan oleh aplikasi untuk menyimpan data dinamis yang dihasilkan oleh sistem.

## Isi Storage

- `logs/`: Berisi file log error aplikasi (`error_log.txt`). Cek folder ini jika ada error blank screen atau error 500.
- `exports/`: Tempat penyimpanan sementara untuk laporan yang diekspor (PDF, Excel).
